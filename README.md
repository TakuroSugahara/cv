## 発揮できるバリュー

新規事業で大切な初期のサービス開発のスピードの重要性を考え、実装しないという捨てる選択を行ってきました。

サーバレス環境や、外部のAPIなどを使い運用コストを下げる取り組みを実施。
出来るだけ開発、運用コストを下げ、スピード開発を意識。
